# Security Policy

Do not commit Discord tokens, Battle.net identifiers, local JSON configs, SQLite databases, or private screenshots.

Supported development happens on `main` plus tagged releases.

Report security-sensitive issues privately to the repository owner instead of opening a public issue. Include:

- A short impact summary.
- Affected component: `D2RHost`, `D2RAgent`, `AgentCommon`, or CI/release.
- Reproduction details with secrets redacted.
- Suggested mitigation, if known.
